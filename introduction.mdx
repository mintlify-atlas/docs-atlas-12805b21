WebBuilder is a Django-based platform that analyses any public API, proposes a site architecture using an LLM, and generates a fully working Django project you can download and run immediately. No templates to pick, no boilerplate to write — the AI does the heavy lifting while you stay in control at every step.

## How it works

<Steps>
  <Step title="Submit an API URL">
    Paste any public URL that returns JSON or XML. WebBuilder fetches the data, detects the format, and safely parses it.
  </Step>
  <Step title="Review the AI plan">
    The LLM analyses the dataset structure and proposes a site type (blog, catalogue, portfolio, dashboard…), relevant fields, and human-readable labels. You can accept the plan or regenerate it with a custom prompt.
  </Step>
  <Step title="Generate the project">
    Once you accept the plan, WebBuilder generates a complete Django project: models, views, HTML templates with Tailwind CSS, a data-loading management command, and a Dockerfile.
  </Step>
  <Step title="Download and deploy">
    Download the project as a ZIP and run it locally, or trigger a one-click deployment via the built-in n8n integration.
  </Step>
</Steps>

## Key features

- **JSON and XML support** — automatic format detection and safe parsing via `defusedxml` + `xmltodict`
- **Multi-model LLM catalog** — Llama 4 Scout (Groq), Llama 3.3 70B and Qwen3 Coder (OpenRouter), or any custom OpenAI-compatible endpoint
- **Iterative plan refinement** — regenerate the AI plan as many times as you like with a natural-language prompt
- **In-browser file editor** — tweak any generated source file before downloading
- **Version history** — every regeneration creates a snapshot you can restore or download at any time
- **n8n automation** — welcome emails and login notifications out of the box
